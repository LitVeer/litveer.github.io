Github SSH 连接错误记录

出现场景

上传文章，执行以下命令时出现
hexo d

解决方法

1.关闭代理

2.将 Github 的连接端口从 22 改为 443

在.ssh下创建文件 config ，内容如下，将连接端口从22改为443

Host github.com
    HostName ssh.github.com
    User git
    Port 443

测试命令 ssh -T git@github.com
